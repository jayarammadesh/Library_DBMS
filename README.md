# 📚 Library Automation System using UiPath

## 📌 Project Overview
This project is a **Library Management Automation System** developed using UiPath (RPA).

It automates key library operations such as:
- Book tracking
- Due date alerts
- Fine calculation
- Smart book location handling

The system reduces manual effort and improves efficiency using automation workflows.

---

## 🚀 Key Features

### 📍 Smart Book Location
- Automatically identifies and manages book locations
- Helps in quick retrieval of books

### ⏰ Due Date Alert System
- Sends alerts for overdue books
- Prevents late returns

### 💰 Fine Calculator
- Automatically calculates fines based on delay
- Reduces manual calculation errors

### 📊 Data Management
- Uses Excel for storing and managing data
- Automates updates and retrieval

### 📧 Notification System
- Sends email alerts using automation workflows

---

## 🛠️ Technologies Used
- **RPA Tool:** UiPath
- **Language:** VB-based workflows
- **Automation:** UI Automation + Excel Automation
- **Mail Integration:** Outlook / SMTP

---

## 🗂️ Project Structure
- `Main.xaml` → Main workflow
- `Smart Book Location.xaml` → Book tracking logic
- `Due Date Alert Sys.xaml` → Alert system
- `Fine Calculator.xaml` → Fine calculation module
- `.json files` → Project configuration

---

## ⚙️ How It Works
1. Input data is read from Excel
2. Workflow processes:
   - Book status
   - Due dates
   - Fine calculation
3. Alerts are triggered if needed
4. Output is updated automatically

---

## ▶️ How to Run
1. Open project in UiPath Studio
2. Install required dependencies
3. Run `Main.xaml`
4. Provide input data (Excel file)
5. Monitor output and alerts
